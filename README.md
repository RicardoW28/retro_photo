Editor de Fotos Retrô 🎞️✨
Um editor de imagens simples e divertido feito com HTML, CSS e JavaScript, que aplica filtros retrô e efeitos visuais inspirados em diferentes épocas e estilos.

-Funcionalidades
Upload de imagens direto do computador.

Aplicação de 12 filtros exclusivos:

Sony Ericsson 2007
Anos 90
VHS
Dreamcore
Vaporwave
Anos 80s
Vintage
Noir
Glitch
Cyberpunk
Nostalgia
CRT TV (com scanlines e aberração cromática)

Botões de ação:

Resetar → limpa a imagem.

Original → restaura a foto ao estado inicial sem filtros.

Baixar Foto → salva a imagem com o filtro aplicado.

Área de GIFs animados no canto inferior direito para dar mais estilo (celular, VHS, CD, dólares, Windows XP, memory card e vinil).

Interface responsiva e organizada, com botões estilizados e efeitos de hover.
